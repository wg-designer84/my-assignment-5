1. What are the different ways to select an element in the DOM?

There are 5 ways to select an element in the DOM: getElementById, getElementsByClassName, querySelectorAll, querySelector, and getElementsByTagName.

2. What is the difference between innerHTML, innerText, and textContent ?

innerText and textContent are almost the same. textContent shows all the text, including hidden text, while innerText shows only the visible text. On the other hand, innerHTML shows all the tags and classes.

3. What is event delegation in the DOM?

Event delegation is a method of handling by adding a single event listener to a parent element. This listener can respond to events that occur on its child elements.

4. What is event bubbling in the DOM?

Event bubbling is a process where an event triggered on an element is passed upwards to its parent elements.

5. How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(), then append it with appendChild(). To remove an element, use removeChild() or parentNode.removeChild().
